# 🚀 Neon Quiz App

A modern, interactive quiz web application built with HTML, CSS, and JavaScript.  
Designed with a premium UI/UX and smooth user experience.

---

## 🌟 Features

✨ Modern Dark UI (Glassmorphism Design)  
⏳ Timer-based quiz system  
📊 Progress bar tracking  
🎯 Multiple categories (Tech, GK, Math, Physics)  
📱 Fully responsive (Mobile + Desktop)  
⚡ Fast and lightweight  
🎨 Clean animations and hover effects  

---

## 🧠 How It Works

- User selects category  
- Questions appear one by one  
- Timer runs for each question  
- Score is calculated dynamically  
- Final result is displayed with performance message
